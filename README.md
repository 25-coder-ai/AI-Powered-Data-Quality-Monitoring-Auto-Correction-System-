# AI Data Quality Monitoring System

An AI-powered system to analyze, monitor, and automatically improve dataset quality using key data quality dimensions.

## 🚀 Features
- Detects missing values, duplicates, and outliers
- Evaluates 6 data quality dimensions:
  - Completeness
  - Accuracy
  - Consistency
  - Validity
  - Uniqueness
  - Integrity
- Provides a Data Quality Score
- Interactive dashboard using Streamlit
- Auto-fix functionality to clean datasets
- Download cleaned dataset

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- NumPy

## ▶️ Run Locally
```bash
streamlit run app.py
